Ruckby
Jersey Store

Site de vente en ligne de maillots et équipements de rugby, développé en React. Boutique avec filtres par catégorie, panier dynamique et récapitulatif de commande.

TypeE-commerce
FrameworkReact + Vite
ContexteProjet personnel
DéploiementVercel

Présentation

Ruckby est un site e-commerce fictif dédié aux maillots officiels et équipements de rugby. Le projet est né de l'envie de combiner deux de mes centres d'intérêt : le développement web moderne avec React et le rugby.

L'application propose une boutique complète avec un catalogue de produits, un système de filtrage par catégorie, un panier dynamique mis à jour en temps réel et un récapitulatif de commande avec calcul du total.

Fonctionnalités

Les principales fonctionnalités développées sur ce projet :

Panier dynamique

Ajout, suppression et modification des quantités en temps réel via le Context API de React.

Filtrage produits

Filtres par catégorie (maillots, équipements, accessoires) mis à jour instantanément sans rechargement.

Résumé commande

Calcul automatique du sous-total, frais de livraison et total TTC dans la page panier.

Interface soignée

Design moderne aux couleurs du rugby, entièrement responsive mobile et desktop.

Stack technique

Le projet s'appuie sur React avec Vite comme bundler pour des performances optimales en développement. La gestion du panier est assurée par le Context API natif de React, sans librairie externe. Le routage est géré par React Router pour la navigation entre la boutique et le panier. Le site est déployé sur Vercel.

React Vite JavaScript Context API React Router CSS Modules Vercel HTML5

Bilan personnel

Ce projet m'a permis de consolider mes bases en React en allant au-delà des tutoriels : concevoir une architecture de composants cohérente, gérer un état global partagé avec le Context API et construire une interface e-commerce fonctionnelle de A à Z.

Le choix du thème rugby était aussi une façon de rendre le projet personnel et motivant. C'est un des premiers projets React que j'ai déployé en production sur Vercel.
